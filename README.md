# Effet Oppos

Predict Metadata from Metagenomic Samples

## Installation

From source
```
git clone git@github.com:Chandrima-04/MetaSUB.git
cd effet_oppos
python setup.py install
```

To run tests
```
cd path/to/effet_oppos
python -m pytest --color=yes . -s
```
